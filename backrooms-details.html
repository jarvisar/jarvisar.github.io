About

This project is inspired by the creepypasta "The Backrooms", which describes a world that is a seemingly endless maze of yellow rooms. To recreate this, I used Three.js to create a procedurally generated maze of eerily empty rooms using primarily Prim's algorithm. The maze infinitely generates as the player moves through it, while the player moves using WASD controls. Infinite procedural generation is achieved by using a system of offset corrdinates and 2D array of mazes, which themselves are 2D arrays of rooms. This can be performance intensive, so the program includes several settingsd to adjust the maze generation and rendering to improve performance.
