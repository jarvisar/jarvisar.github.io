About

Angular-powered web app capable of visually representing a wide range of different sorting algorithms, such as insertion sort, merge sort, heap sort, radix sort, bitonic sort, and even stooge sort. The app also allows the user to set the size of the array to be sorted and change the speed of the sorting process.

Also available as an APK for Android.

Check out the GitHub repo to view the list of supported algorithms.
